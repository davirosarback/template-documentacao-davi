Tempalte para criação de documentação de manuais em *portugues* e *espanhol*, utilizar o **Nentra** JS v3.

Para utilizar esse template, basta cloná-lo e executar o comando `npm instal`. Após instalação, executar o comando `npm run dev` para rodar o servidor de desenvolvimento local.

A rota inicial padrão para o idioma portguês é `localhost:3000/pt` e par o espanhol é `localhost:3000/es`.
